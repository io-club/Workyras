Instruction

Operations

Load

You select a directory in which work_tags.csv and work_points.csv must be.

App will load data from the two files.

Save

You select a directory which must be empty.

App will save data to work_tags.csv and work_points.csv in the directory.

Format details

Following files are in CSV format, using comma as delimiter.

Empty fields will be interpreted as null.

work_tags.csv

ID ID_GLOBAL NAME IS_RECYCLED NEXT_ID
1 00112233445566778899aabbccddeeff My Tag 1 false 2
2 10112233445566778899aabbccddeeff My Tag 2 false
(Long) (String) (String) (Boolean) (Long, Nullable)
  • ID_GLOBAL: Global work tag ID in hex string.
  • NEXT_ID: Next tag's local ID. null if it's the last.

work_points.csv

WORK_TAG_ID COMMENT PRODUCED_AT
1 1735689600000
(Long) (String, Nullable) (Long)
  • COMMENT: null if no comment (typically).
  • PRODUCED_AT: Timestamp, milliseconds since epoch.
